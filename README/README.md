\# 🖐️ Gestures Control



Управление компьютером через жесты рук с веб-камеры.

\- ✋ Движение курсора указательным пальцем

\- ✊ Левый клик (кулак)

\- 🤘 Правый клик (рокер)

\- ✌️ Скроллинг двумя пальцами

\- 🧠 Управление головой (для Minecraft)
