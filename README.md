# GesturesControl
Gestures Control — управление ПК жестами рук через веб-камеру. Указательный палец = движение курсора, кулак = ЛКМ, рокер = ПКМ, два пальца = скроллинг. Отдельный режим для Minecraft: поворот головы, ходьба, прыжки. Пригодится, когда мышь плохо работает или просто ради фана. Работает на любой камере. 100+ FPS, автокалибровка. Python OpenCV MediaPipe
